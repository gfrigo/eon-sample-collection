# Processamento de imagens e captura de vídeo
opencv-python==4.9.0.80

# Controle de portas GPIO do Raspberry Pi
RPi.GPIO==0.7.0

# Biblioteca para controlar LCD I2C 16x2 (PCF8574)
RPLCD==2.0.0

# ─────────────────────────────────────────────────────────────────────────────
# Observações:
# 
# • opencv-python: Versão fixa 4.9.0.80 é a mais recente estável testada
#   com Raspberry Pi 4 (2GB RAM). Se tiver problemas de memória durante
#   compilação, tente a versão pre-built ou adicione mais swap.
#
# • RPi.GPIO: Versão 0.7.0 é a mais estável para RPi 4 com Raspberry Pi OS.
#   Só funciona em um Raspberry Pi real (não em simuladores Linux genéricos).
#
# • RPLCD: Versão 2.0.0 suporta Python 3.7+ e é bem testada com PCF8574.
#   Se usar em um modelo antigo de LCD, tente versão 1.3.1 como fallback.
